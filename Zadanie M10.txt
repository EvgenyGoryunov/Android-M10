Есть файл формата .py или .java (файлы, написанные на языке программирования python и java), которые содержат в
себе фрагменты кода.
Нужно написать программу и упаковать ее с .jar расширением:

Данная программа производит массовое экранирование, либо разэкранирование принтов, то есть для отладки кода удобно
использовать функции print в коде для понимания порядка исполнения.

Программа анализирует тип файла (по расширению) и в зависимости от этого производит соответствующие действия
для конкретного типа кода (либо py, либо java)

Также программа сама определяет состояния принтов в коде, либо все принты экранированные - программа разэкранирует их,
либо наоборот.
